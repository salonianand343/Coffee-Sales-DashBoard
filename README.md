# ☕ Coffee Shop Sales Dashboard 📊

An interactive **Coffee Shop Sales Dashboard** built using **Microsoft Excel** to analyze sales performance, customer behavior, product trends, and store-wise insights. This project transforms raw transactional data into meaningful business intelligence using Pivot Tables, Pivot Charts, slicers, and KPI visualizations.

---

# 🚀 Project Overview

The dashboard provides a complete analysis of coffee shop operations by tracking:

- Total Sales Revenue
- Customer Footfall
- Average Bill per Customer
- Average Orders per Customer
- Product & Category Performance
- Store-wise Sales Analysis
- Hourly & Weekday Order Trends

The project helps businesses make data-driven decisions through interactive and visually appealing reports.

- Top-selling product analysis
- Store location performance comparison
- Category and size distribution analysis
- Hourly customer ordering trend visualization

---

# 📈 Key Insights

- Coffee products contributed the highest sales revenue.
- Peak customer orders were observed during morning and afternoon hours.
- Hell’s Kitchen store generated the highest sales.
- Regular and Large sizes were the most preferred order sizes.
- Barista Espresso emerged as the top-selling product.
